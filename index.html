Project 1 (Jan→Feb)

Take for instance our train vibration measurement system which clocks up to speeds of 168MHz measuring vibration on all 3 XYZ axis using true state-of-the-art nanotech (MEMS sensor) - everything is done in pure real time writing data to a 32 Gig SD card up to speeds of 7 mega bytes per seconds and mapped by a FAT32 file system! It's an exciting achievement for us.


That's one months worth of vibration measurement on a small battery powered device!!

For our vibration sensor, we want to be smaller, lower power, and be able to do more!
